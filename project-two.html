IoT Flowerpot

App for a Flowerpot System

This project consists of two parts build a flowerpot system and mobile app for iOS and Android using Java, Kotlin, C++ for Arduino

The app connects to an Arduino in the flowerpot system and sends commands to control and read from different peripherals attached to the flowerpot in real time.

The flowerpot included the following components:

  • Arduino
  • Thermometer
  • Temperature
  • pH scale
  • Humidity
  • Water pump
  • Light sensor
